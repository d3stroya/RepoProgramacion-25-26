# Red social de gamers 🎮

Un estudio de videojuegos nos ha encargado un software 
que gestione los jugadores de su plataforma.

Los jugadores se almacenarán en un mapa con clave ID y valor Jugador.

Cada jugador contiene un conjunto de juegos.

La clase Jugador tiene como atributos:

* ID (Integer)
* nickname (String)
* edad (int)
* juegos favoritos (Set)

## Funcionalidades
Mostraremos por consola un menú que permita:

1. Añadir jugador
2. Añadir juego favorito a un jugador
3. Mostrar todos los jugadores
4. Buscar jugador por ID
5. Filtrar jugadores por juego
6. Eliminar jugador
7. Mostrar juego más popular
8. Recomendar jugadores con gustos similares
